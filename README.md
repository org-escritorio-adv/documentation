# Escritório de Advocacia

## Descrição do Projeto
Este projeto tem como objetivo o desenvolvimento de um sistema web para um escritório de advocacia, composto por:

- Área institucional (site público)
- Área restrita para advogados
- Integração com API jurídica (consulta de processos)
- Gestão interna de processos e clientes

O sistema visa otimizar o acompanhamento processual e centralizar informações relevantes para o escritório.

---

## Objetivos
- Facilitar a consulta de processos judiciais
- Centralizar dados de clientes e processos
- Melhorar a produtividade dos advogados
- Garantir segurança e organização das informações

---

## Funcionalidades Principais
- Autenticação de usuários
- Consulta de processos via API
- Cadastro e gestão de clientes
- Organização de processos internos
- Dashboard com informações relevantes
- Interface responsiva

---

## Arquitetura do Projeto
O sistema será estruturado em:

- Backend
- Frontend
- Banco de dados
- Microserviço de autenticação
- Integrações externas (API jurídica)

---

## Tecnologias (a definir/ajustar)
- Backend: Python
- Frontend: FastAPI
- Banco de Dados: Postgres
- Containerização: Docker
- Testes: Selenium + testes unitários/integrados
